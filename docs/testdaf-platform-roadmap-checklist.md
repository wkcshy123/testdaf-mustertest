# TestDaF 模拟考试系统总体路线图 Checklist

## 当前定位

- [x] 建立本地优先的 TestDaF 模拟考试 Web 系统。
- [x] 完成老师端出题系统，覆盖听力、阅读、写作、口语四大模块。
- [x] 所有生成题目、文本、音频、图表、上传素材与预览文件保存到本地 `question_bank/`。
- [x] 使用 `qwen3.7-plus` 作为非 TTS 生成模型，听力/口语语音继续使用 Qwen-TTS。
- [x] 支持文字参考素材、网页参考素材，写作和口语图表题支持本地图片参考素材。
- [ ] 建立完整学生答题系统，支持抽题、作答、计时和答案保存。
- [ ] 建立评分系统，支持客观题自动评分、写作/口语辅助评分与老师复核。

## 总体技术路线

- [x] 采用 FastAPI + Jinja2 服务端模板快速成型。
- [x] 采用模块化单体架构，按 `services/`、`storage/`、`templates/`、`static/` 分层。
- [x] 使用 `uv` 进行 Python 包管理。
- [x] 使用本地文件系统作为早期题库主存储。
- [x] 使用 `QuestionBank` 统一保存题目包、manifest、预览和资源文件。
- [x] 使用后台 `JobManager` 支持口语单题并行生成。
- [ ] 后续题库规模扩大后，可增加 SQLite 作为索引层。
- [ ] 后续根据复杂度，将听、读、写、说模块逐步拆分为独立服务。

## 老师端出题模块

- [x] 老师出题索引页：`/teacher`。
- [x] 听力 Aufgabe 1：校园双人对话，8 道短答题，分段多音色 TTS。
- [x] 听力 Aufgabe 2：主持人与两位嘉宾/专家访谈，10 道 Richtig/Falsch。
- [x] 听力 Aufgabe 3：主持人与专家访谈，7 道短答题，支持二次播放元数据。
- [x] 阅读 Aufgabe 1：A-H 短文本与 10 个人物需求匹配，含无匹配项 `I`。
- [x] 阅读 Aufgabe 2：中长说明/科普文本，10 道 A/B/C 单选题。
- [x] 阅读 Aufgabe 3：长阅读文本，10 道 Ja/Nein/Text sagt dazu nichts。
- [x] 写作 Aufgabe 1：短题干、核心提问、任务要求、1-2 张生成图表。
- [x] 口语 Aufgabe 1：电话咨询/信息询问，引子语音。
- [x] 口语 Aufgabe 2：个人或本国情况说明，引子语音。
- [x] 口语 Aufgabe 3：图表描述，支持参考图片、生成图表和引子语音。
- [x] 口语 Aufgabe 4：复杂议题立场表达，引子语音。
- [x] 口语 Aufgabe 5：学习/人生决策建议，引子语音。
- [x] 口语 Aufgabe 6：图表 + 原因/影响分析，支持参考图片、生成图表和引子语音。
- [x] 口语 Aufgabe 7：复杂生活化选择建议，引子语音。
- [ ] 支持老师编辑 AI 生成结果后再保存。
- [ ] 支持题目版本管理、标签管理和正式发布状态。

## 素材与生成能力

- [x] 所有题型支持文字参考素材。
- [x] 所有题型支持多个 HTML 网页链接作为参考素材。
- [x] 网页参考素材会提取标题和正文摘要，并保存 `reference_sources.json`。
- [x] 写作题支持多张本地参考图片，多模态生成题干和图表规格。
- [x] 口语图表题 `Aufgabe 3/6` 支持本地参考图片，多模态生成题目和图表规格。
- [x] 听力题支持结构化分段、多说话人音色映射、停顿插入和 WAV 拼接。
- [x] 口语题支持每题单独生成引子语音。
- [x] 阅读题支持 UTF-8 长度约束和失败后的 LLM 修复。
- [ ] 可选增强：建立本地环境音素材库，为听力题混入低音量背景音。

## 本地题库结构

- [x] 自动创建 `question_bank/listening`、`reading`、`writing`、`speaking`。
- [x] 每个题目包保存 `manifest.json`。
- [x] 每个题目包保存 `preview.md`，老师端用 Markdown 渲染预览。
- [x] 听力题保存 `transcript.txt`、`segments.json`、`questions.json` 或 `statements.json`、`audio.wav`、`audio_segments/`。
- [x] 阅读题保存 `texts.json`、`profiles.json`、`reading_text.txt`、`paragraphs.json`、`questions.json` 或 `statements.json`。
- [x] 写作题保存 `prompt.json`、`charts.json`、`chart_*.svg`、`reference_images/`。
- [x] 口语题保存 `prompt.json`、`charts.json`、`chart_*.svg`、`intro.wav`、可选 `reference_images/`。
- [x] 题目包保存 `reference_sources.json` 以追溯网页参考素材。
- [ ] 后续统一资源目录规范，例如所有素材集中到 `assets/`。
